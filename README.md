# Java Interviews Preparation

A comprehensive learning resource for Object-Oriented Programming (OOP) concepts in Java, designed to help developers prepare for technical interviews and strengthen their core Java fundamentals.

## 📚 Overview

This repository contains well-documented Java examples covering essential OOP concepts with practical implementations and explanations. Each module demonstrates key interview topics with clean, production-ready code.

## 🎯 Learning Objectives

- Master fundamental OOP principles in Java
- Understand constructors, inheritance, and object relationships
- Prepare for Java-based technical interviews
- Learn best practices and design patterns

## 🔍 Topics Covered

### Chapter 1: Constructors & Inheritance

1. **Default Constructor** - Understanding default initialization
2. **Parameterized Constructor** - Object creation with initial values
3. **Constructor Chaining** - Using `this()` keyword
4. **Super Keyword** - Parent class method and constructor access
5. **Inheritance** - Extending classes and method overriding

## 💻 Example Usage

